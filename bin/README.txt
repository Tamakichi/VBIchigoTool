IchigoJam支援ツール GUI版 1.03 簡易マニュアル　

マニュアル作成日  2015/05/24
マニュアル作成日  2019/11/07
作成者                たま吉さん

はじめに
このマニュアルは、IchigoJamを支援するためのコマンドの簡易解説マニュアルです。
本コマンドを使うことによりIchigoJam上のプログラムをパソコンに保存したり、
パソコン上のプログラムをIchigoJamに読み込むことが出来ます。

利用環境
・IchigoJam本体
・パソコン  OSとして Windows 10 搭載機
・パソコンとIchigoJamでシリアル通信が出来ること(115,200bps)
・本プログラムはMicrosofit Visual Studio Express 2019 環境にて
   Visual Basicにて開発しています。別途ランタイムモジュールが必要です。
    フレームワーク .NET Framework 4.0以上  

配布物
  Ichigotool.exe              プログラム本体
  README.txt                  この説明書です

インストール 
  ・インストール作業は不要です。お好きなフォルダに保存して利用いて下さい。
  ・環境によては、ランタイムモジュール等のインストールが必要となります。
   
  ・インターネットからダウンロードした場合、セキュリティブロックが
     かかっている場合があります。
     Ichigotool.exeのプロパティを開いてセキュリティのブロックを
     解除してください。

使い方
    事前に、IchigoJamとパソコン間でシリアル接続をして下さい。
    Ichigotool.exe を実行します。

  ・IchigoJamにプログラムをアップロードする
     1)シリアルポート を選択します。
     2)[アップロード]ボタンをクリックします。
     3)ファイル選択画面でアップロードするファイルを選択します。
         選択後、[開く]ボタンをクリックします。
     4)確認画面で[はい]をクリックすると、アップロードを開始します。
     5)アップロードが完了すると、完了を通知する画面が表示されます。
     
     アップロードを中止したい場合は、[中止]ボタンをクリックして下さい。
     アップロードを中止します。
  
  ・IchigoJamからプログラムをダウンロードする
     1)シリアルポート を選択します。
     2)[ダウンロード]ボタンをクリックします。
     3)「名前を付けて保存]画面で保存するファイルを指定します。
     4)確認画面で[はい]をクリックすると、ダウンロードを開始します。
     5)アップロードが完了すると、完了を通知する画面が表示されます。
        メモ帳で開く選択を行うと、メモ帳を起動してプログラムを表示します。
